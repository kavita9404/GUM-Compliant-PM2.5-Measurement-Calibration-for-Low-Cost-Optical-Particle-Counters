# GUM-Compliant-PM2.5-Measurement-Calibration-for-Low-Cost-Optical-Particle-Counters
# PM2.5 Calibration using Hybrid ML + GUM Uncertainty

##  Overview
This repository contains code for calibrating low-cost PM2.5 sensors using machine learning models (XGBoost, LightGBM, Random Forest) with GUM-compliant uncertainty quantification.

##  Dataset
QUANT dataset:
https://doi.org/10.1038/s41597-024-03767-2

##  How to Run
1. Open notebook in Google Colab
2. Run all cells
3. Results will reproduce figures from the paper

##  Models Used
- XGBoost
- LightGBM
- Random Forest
- Ensemble (NNLS)
